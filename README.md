## ⚙️ SETUP

git clone https://github.com/ankur6000/SMARTLOCAL_ONLINESTORE_BUSINESSPANEL365.git

cd SMARTLOCAL_ONLINESTORE_BUSINESSPANEL365
npm install


Create `.env`

PORT=3000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret


Run:

npm start

## 🚀 DEPLOYMENT (RENDER)
